# Architecture: 05-portfolio-site-v2

## Overview
A dark neon-themed portfolio and course template built with React 19 and Vite 6. The architecture emphasizes high-fidelity animations and a structured data-driven approach to content.

## Tech Stack
-   **Framework**: React 19
-   **Build Tool**: Vite 6
-   **Styling**: Tailwind CSS v4
-   **Animations**: Framer Motion 12
-   **Icons**: Lucide React

## Key Components
-   **Hero Section**: Immersive dark background with radial pulse effects and gradient typography.
-   **Concepts Grid**: Staggered animation grid for core technical features.
-   **Enrolment Form**: Structured feedback and enrolment interface with interactive states.

## Performance
-   Tree-shaken dependencies with Lucide React.
-   Optimized Framer Motion variants for reduced main-thread load.
-   Vite-optimized asset pipeline.
