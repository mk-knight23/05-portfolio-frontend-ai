# Design System: 05-portfolio-site-v2

## Visual Philosophy
The design follows a **Dark Neon / Cyber-Premium** aesthetic. It combines deep slate backgrounds with vibrant indigo and purple glow effects to create a high-contrast, professional technical feel.

## Design Patterns
-   **Glow Effects**: Radial blurs used for background depth and interactive highlights.
-   **Typography**: Variable weight sans-serif (Inter/Default) with extreme tracking for headers.
-   **Neon Accents**: Indigo (#6366f1) and Purple (#a855f7) gradients.

## Color Palette
-   **Base Surface**: Slate Black (#020617)
-   **Accent Glow**: Indigo-500/10
-   **Typography High**: Pure White (#FFFFFF)
-   **Typography Low**: Slate-400 (#94a3b8)

## Interaction Design
-   **Staggered Entrances**: Viewport-triggered entrances for the concepts section.
-   **Pulse Animations**: Continuous subtle background pulses to create a "living" interface.
-   **Neon Hover States**: Intensified borders and shadows on interactive cards.
